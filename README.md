## 📖 项目简介

**寿生宝鉴**是一款基于传统道教文化与现代Web技术结合的阴债计算工具。根据用户的出生年月日时（八字），精准推算受生欠债数额，包括：

- **官债**（欠地府曹官之债）
- **私债**（欠冥司私债）
- **利息**（按年计算的利息累积）
- **读经遍数**（还债所需诵读经文遍数）
- **纳库数目**（应纳天库、地库数目）
- **曹官信息**（掌管该命造的曹官姓名）

本工具仅供文化研究、命理学习参考，请勿迷信。

## ✨ 功能特色

| 功能 | 说明 |
|------|------|
| 🔮 精准八字推算 | 支持农历/公历输入，自动计算年月日时四柱 |
| 📊 详细欠额明细 | 官债·私债·利息逐项列明，总计一目了然 |
| 📜 历史记录保存 | 登录后每次计算自动保存，随时查询历史 |
| 📖 读经遍数指引 | 根据年柱给出所需读经遍数与纳库数 |
| 🌙 农历闰月支持 | 完整支持闰月出生者的精确计算 |
| ⏰ 十二时辰对应 | 子丑寅卯辰巳午未申酉戌亥完整对应 |
| 👤 用户系统 | 注册/登录/个人中心/修改密码/注销账号 |
| 📱 响应式设计 | 完美适配桌面端和移动端 |

## 🚀 快速开始

### 本地运行

1. **克隆项目**
```bash
git clone https://github.com/[your-username]/yinzhai.git
cd yinzhai
```

2. **安装依赖**
```bash
npm install
```

3. **启动本地服务器**

方式一：使用自带的 `start.bat`（Windows）
```bash
双击 start.bat
```

```bash
python server.py
```

方式三：使用 Node.js
```bash
npx serve .
```

浏览器打开 `http://localhost:8000`（端口根据实际情况调整）

## 🔑 使用方法

### 1. 开始计算（无需登录）

访问首页 → 点击「开始计算」→ 输入出生信息（支持农历/公历切换）→ 点击「开始计算」→ 查看详细结果

### 2. 注册账号（保存历史记录）

点击「注册」→ 填写用户名、密码、手机号（选填）、邮箱（选填）→ 注册成功 → 登录

### 3. 查看历史记录

登录状态 → 首页点击「查看历史」或顶部导航「历史」→ 浏览所有计算记录 → 支持关键词搜索

### 4. 个人中心

登录状态 → 顶部导航「个人中心」→ 查看个人信息 → 修改密码 / 注销账号
