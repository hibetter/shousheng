# 寿生宝鉴 - 安装与使用指南

## 系统要求

- **操作系统**：Windows 10/11 或 macOS/Linux
- **Python**：3.7 或更高版本（Python 3.13 已测试通过）
- **浏览器**：Chrome、Edge、Firefox 等现代浏览器

---

## SQLite 安装说明

本系统使用 SQLite 作为本地数据库。SQLite 是 Python 内置支持的，无需额外安装！

### Windows 用户

Python 3 已经自带 SQLite，无需任何额外安装。

验证方法：
1. 打开命令提示符（Win + R，输入 `cmd`，回车）
2. 输入以下命令验证：
```
python -c "import sqlite3; print('SQLite 版本:', sqlite3.sqlite_version)"
```

### macOS 用户

macOS 自带 Python 和 SQLite，无需额外安装。

### Linux 用户

大多数 Linux 发行版自带 Python 和 SQLite。如果提示没有 sqlite3，请运行：

**Ubuntu/Debian:**
```bash
sudo apt install python3 sqlite3
```

**CentOS/RHEL:**
```bash
sudo yum install python3 sqlite
```

## 快速启动

### 1. 启动服务

双击运行项目目录中的 `start.bat` 文件，或者在命令行中运行：

```bash
cd yinzhai
start.bat
```

或者手动启动：

```bash
cd yinzhai
python server.py
```

### 2. 访问系统

打开浏览器，访问：**http://localhost:5000**

### 3. 使用流程

1. **注册账号**：点击"注册"按钮创建账号
2. **登录**：使用账号登录
3. **计算阴债**：输入出生信息，计算阴债
4. **保存记录**：登录后可保存计算结果
5. **查看历史**：在"历史记录"中查看所有保存的计算

## 功能说明

### 债务计算逻辑

总计 = **官债** + **私债** + **官债利息** + **私债利息**

- **官债**：年柱 + 月干 + 月份 + 日柱 + 时干 + 时支
- **私债**：= 官债
- **官债利息**：= 私债利息 = 私债 × 年龄倍数
- **年龄倍数**：0-9岁×2，10-19岁×3，20-29岁×4，以此类推

### 数据存储

- 数据库文件：`yinzhai.db`（SQLite 数据库）
- 数据库表：
  - `users`：用户信息表
  - `records`：计算记录表

### 数据备份

数据库文件就是 `yinzhai.db`，直接复制备份即可。

## 常见问题

### Q: 启动后浏览器无法访问？

A: 确认命令行中没有错误信息。检查端口5000是否被占用。

### Q: 如何修改端口？

A: 编辑 `server.py` 文件，找到最后一行的 `port=5000`，修改为其他端口（如8080）。

### Q: 如何重置数据库？

A: 关闭服务后，删除 `yinzhai.db` 文件，重新启动服务即可。

### Q: 注册时提示"用户名已存在"？

A: 该用户名已被注册，请使用其他用户名。
